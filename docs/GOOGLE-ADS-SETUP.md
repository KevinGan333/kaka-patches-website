# Google Ads Setup — KaKa Patches

## Conversion Tracking

| Setting | Value |
|---|---|
| Conversion action | Quote form submitted |
| Trigger page | `/thank-you` |
| Conversion type | Lead / Submit form |
| Count | One per click |

Place the Google Ads conversion snippet on `/thank-you` (via `<Script>` in Next.js or via Google Tag Manager).

## Campaign Structure

### Campaign: Custom Patches — Search

**Budget:** Start at $30–50/day, scale based on conversion data.

| Ad Group | Keywords (exact/phrase) |
|---|---|
| Custom Patches | "custom patch manufacturer", "custom patches bulk", "custom patch supplier", "B2B custom patches" |
| Embroidered Patches | "custom embroidered patches", "embroidered patch manufacturer", "embroidered patches bulk", "custom embroidery patches" |
| Woven Patches | "custom woven patches", "woven patch manufacturer", "custom woven labels", "woven patches bulk" |
| PVC Patches | "custom PVC patches", "PVC patch manufacturer", "custom rubber patches", "PVC patches bulk" |
| Chenille Patches | "custom chenille patches", "chenille patch manufacturer", "varsity letter patches", "chenille patches bulk" |

## Recommended Audiences

- **Location:** US, Canada, UK, Australia, EU
- **Language:** English
- **Device:** All
- **Network:** Google Search (not Display, not Partners initially)

## Ad Copy Template

Headline 1: **Custom {Patch Type} Manufacturer**
Headline 2: **Factory-Direct Bulk Pricing**
Headline 3: **Free Artwork Review — Quote Today**
Description: Factory-direct custom patch production for B2B buyers. Bulk orders from 100 to 100,000+ pieces. Free artwork review and fast quote response.
